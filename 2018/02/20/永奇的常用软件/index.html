实用的60个CSS代码片段

发表于 2018-02-20 | 分类于 apps

使用电脑手机多了，每个人都会有自己的使用习惯，会有自己的常用软件，在这里我分享下自己的常用软件。

手机-iPhone/Android

iOS邮箱

选择自带原因是与iOS的联动最好。比如点击邮件里的地址就会跳到地图。并规划路线。点击日期时间就会创建事件

网易邮箱大师

这段时间翻墙不稳定，而我又有不少东西绑定在了 gmail 邮箱，网易邮箱正好可以不翻墙收发 gmail ，正好满足了我的需求

网易云音乐

多端同步。挺好用的。这个看大家的习惯和账号体系在谁家。体验差不多。而且曲库也都差不多。现在好像是腾讯系的资源比阿里系好点？
听说网易和腾讯交叉授权啦?

seed-国际新闻

这个软件是英文新闻app。点击即可查看翻译。非常适合学习英语。

即刻

新闻聚合类app。适合喜欢阅读新闻的人。喜欢看什么内容。订阅就好。交互也不错。内容也好。

appso

少数派专门收集app的地方。对新奇app有兴趣的可以看看。交互不错。也有微信订阅号可以关注。

workflow

IOS专用。这个厉害了。现在免费了。可以自动化任务。网上有不少编好的直接下载拿来用。也可以自己写规则。不难。

幕布

这个东西就是用幕布写的。转到网上稍微编辑一下就可以用啦！

Keep

健身app，用过的人自然懂，目前做的最好的健身app

fooView

Android专用。系统增强app，可以帮你更快更简单的处理图片文字截图等。

酷安

Android专用。Android应用商店。建议刚用Android手机的来这里看看，总能发现你需要的。

PC-Windows 10

Listary-Windows 下快速启动工具

快速启动工具，相当与 Mac 的 Spotlight，可以快速启动软件或者搜索文件，生产力工具，当然不止这些功能，之前在用 Wox 的，可是不知道出了神马问题无法使用了，于是试了下其他的快速启动工具，选了这款，国人制作，免费，出色的新手引导，基本不用额外设置，好评！

PS：现在又用回了 Wox 了，毕竟 UI 更合我的口味。这两个选那个都可以，差别不大。

Seer-Windows 下预览工具

还是源于 Mac 的功能，选中文件，按下空格预览，这里与Mac用法相同，效果也很好。
PS：目前在用 QuickLook，也是同功能软件，选择那个看个人喜好。

Snipaste-Windows 下截图工具

非常好用，如果对截图要求不高，Widnows 10 自带的截图快捷键 Win+Shift+S 键也可以用，摁下直接截图，并且保存到复制版，放在QQ或者微信直接

ADSafe-非常强大的广告拦截软件

因为太强大了，所以有被监守自盗的风险，对电脑安全要求高的谨慎使用，个人支付都在手机完成，so，用这个非常省心好用

Potplayer-本地视频播放器

经常下载一些电影，美剧看，这款播放器不需要太多设置即可食用，本身的设置也很足，大多数人使用足够了。

FreeDownload-代替迅雷的下载软件

我也是刚刚使用，正在试毒，界面比较讨喜，无广告，下载速度也可以，但是迅雷有P2P。建议大家试试把。

Typora-MarkDown 编辑器

使用这款因为颜值高，免费，推荐大家学学MarkDown 语法，不难，但是效率真心高了不少。

TIM-QQ企业版

之前的轻聊版，现在叫TIM，干净的QQ。

Chrome-浏览器

浏览器简介不写了，这里推荐我正在使用的插件

Google翻译

好像不翻墙也能用? Google自家产品，非常好用，也可以翻译整个网页。

Stylish-为任意网站自定义主题

不喜欢网站样式，可以换一种，可以自己写，也可以用其他人分享的，我分享一下我正在用的，用法很简单，跟着点就可以了，我就不写了

Baidu Lite 百度轻
Douyu Cleaner 清爽斗鱼
Flat_Zhihu
weibo_v6

简悦

阅读模式，国人的软件当然最了解国人使用习惯，这款为大部分网站都有适配，颜值高，我认为大家都应该有一个阅读模式的插件

JSON-handle 格式化 JSON ,程序员都懂。 * Octotree 为 GitHub 提供一个文件数，方便切换，如果长逛的话 GitHub，强烈建议。

SlideToShutDeown-Windows 关机

最后这个不是神马软件，是windows 自带的关机程序，运行后，会从上出现开机画面到中间，然后往下拉这个画面到下面就关机了，多了点逼格， 文件位置在 C:\Windows\System32/SlideToShutDown.exe
